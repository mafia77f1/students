

# منصة ستدي زون (StudyZone) - خطة التطوير

## الرؤية
منصة مذاكرة جماعية ملعّبة بتصميم عصري داكن مع ألوان بنفسجية ونيون أزرق، واجهة عربية بالكامل.

## الصفحات والمكونات

### 1. لوحة التحكم (Dashboard)
- بطاقات إحصائيات: ساعات المذاكرة، المستوى (Level)، الرتبة (Rank)، XP
- شريط تقدم XP مع تأثيرات بصرية متوهجة
- لوحة الصدارة الأسبوعية (أفضل 10 طلاب)
- ملخص النشاط الأخير

### 2. صالة الغرف (Lobby)
- قائمة غرف الدراسة النشطة مع عدد المتواجدين
- فلترة حسب التخصص (هندسة، طب، لغات، علوم، أخرى)
- زر إنشاء غرفة جديدة
- بحث في الغرف

### 3. غرفة الدراسة (Study Room)
- مؤقت Pomodoro مركزي (25 دقيقة دراسة / 5 دقيقة استراحة)
- قائمة المتواجدين مع حالاتهم (يذاكر 🟢 / في استراحة 🟡)
- صندوق دردشة جانبي (Live Chat)
- قائمة مهام شخصية (To-Do List) مع تحديد الإنجاز
- زر "تحدي" لبدء سباق مذاكرة سريع
- تنبيهات صوتية/بصرية عند انتهاء الوقت

### 4. لوحة الصدارة (Leaderboard)
- ترتيب أسبوعي لأفضل 10 طلاب
- عرض الرتبة والـ XP لكل طالب

### 5. صفحة الملف الشخصي (Profile)
- عرض الرتبة الحالية مع شارة بصرية
- إحصائيات تفصيلية

## نظام التلعيب
- **الرتب**: Bronze → Silver → Gold → Platinum → Diamond → Grandmaster
- شريط XP يزداد مع تشغيل المؤقت
- شارات وأيقونات لكل رتبة

## التصميم
- **Dark Mode** كأساس مع تدرجات بنفسجية (#8B5CF6) ونيون أزرق (#06B6D4)
- خط **Tajawal** للواجهة العربية بالكامل (RTL)
- أنيميشن بين الصفحات باستخدام Framer Motion
- مكونات shadcn/ui مخصصة بالثيم الداكن
- تصميم متجاوب بالكامل (جوال + حاسوب)

## الملاحظة
هذا الإصدار سيعمل ببيانات محلية (Mock Data) بدون قاعدة بيانات. يمكن ربطه بـ Supabase لاحقاً للمصادقة والبيانات الحية والتزامن.

## الهيكل التقني
- React Router للتنقل بين الصفحات
- Sidebar للتنقل الرئيسي
- Framer Motion للأنيميشن
- حالة محلية (useState/useContext) لإدارة البيانات
- نظام تنبيهات باستخدام toast notifications
